一、前端构建步骤

1. 安装 Node.js

  • 访问官网下载安装： https://nodejs.org/zh-cn/download
  • 下载 Windows 64 位 .msi 文件，安装时建议不要安装到 C 盘或带中文的目录。

验证安装是否成功：

node -v
npm -v

2. 升级相关依赖 

npm install express --registry=https://registry.npm.taobao.org
npm install -g node
npm install -g npm
npm install -g cnpm --registry=https://registry.npm.taobao.org

3. 安装 vue-cli 脚手架工具 

npm install -g @vue/cli
vue -V  # 查看版本，必须为 4.5.X 以上

如需升级 vue-cli：

npm update -g @vue/cli

卸载命令：

npm uninstall -g vue-cli

4. 创建 Vue3 项目 

cd your_project_path  # 进入项目目录（最好使用英文路径）
vue create 项目名

5. 启动项目 

cd 项目名
npm run serve

6. 使用 VSCode 打开项目，参考网上 Vue 项目目录结构说明

二、后端服务端构建（Flask + MySQL）

1. 准备环境

  • 安装 MySQL 数据库
  • 安装 Python 3.7
  • 安装 PyCharm 专业版

2. 创建 Flask 项目

  1. 打开 PyCharm，新建项目
  2. 左侧选择 Flask，指定项目目录

3. 编辑 app.py 文件，粘贴如下服务端代码：

导入依赖：

from flask import Flask, request
import pymysql
import json

app = Flask(__name__)

接口1：获取所有数据

@app.route('/getBrowserByDate1', methods=['GET'])
def getBrowserByDate1():
    conn = pymysql.connect(host='127.0.0.1', user='root', passwd='123456', db='shtd_store', charset='utf8')
    cur = conn.cursor()
    cur.execute("SELECT * FROM tb_browsers")
    res = cur.fetchall()
    cur.close()
    conn.commit()
    conn.close()

    arrayResult = []
    for row in res:
        result = {
            'id': row[0],
            'user': row[1],
            'code': row[2],
            'browser': row[3],
            'banner': row[4],
            'score_designer': row[5],
            'score_speed': row[6],
            'score_content': row[7],
            'visit_date': row[8]
        }
        arrayResult.append(result)

    result = {
        'code': 200,
        'msg': '查询成功',
        'results': arrayResult
    }
    return json.dumps(result).encode('utf-8').decode('unicode_escape')

接口2：按时间段查询数据

@app.route('/getBrowserByDate2', methods=['POST'])
def getBrowserByDate2():
    start = request.form.get('start')
    end = request.form.get('end')
    conn = pymysql.connect(host='127.0.0.1', user='root', passwd='123456', db='shtd_store', charset='utf8')
    cur = conn.cursor()
    cur.execute("SELECT * FROM tb_browsers WHERE visit_date >= '{}' AND visit_date <= '{}'".format(start, end))
    res = cur.fetchall()
    cur.close()
    conn.commit()
    conn.close()

    arrayResult = []
    for row in res:
        result = {
            'id': row[0],
            'user': row[1],
            'code': row[2],
            'browser': row[3],
            'banner': row[4],
            'score_designer': row[5],
            'score_speed': row[6],
            'score_content': row[7],
            'visit_date': row[8]
        }
        arrayResult.append(result)

    result = {
        'code': 200,
        'msg': '查询成功',
        'results': arrayResult
    }
    return json.dumps(result).encode('utf-8').decode('unicode_escape')

4. 依赖安装

如果出现 pymysql 或 json 模块导入失败，请使用以下命令安装依赖：

pip install PyMySQL -i https://pypi.tuna.tsinghua.edu.cn/simple

至此，前后端项目搭建完毕。
